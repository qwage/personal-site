# Continued Education

## Python Programing :snake:

```py
from uw import python
```

After working in the [NVE organization](../work_experience/boeing.md#noise-vibrations-and-emissions), I
realized the power and efficiency that was possible using programing with engineering knowledge. The NVE organization
was using Python heavily and I wanted to be a part of creating the future tools for the group. With the advisement of
[Jesse Follet](https://www.linkedin.com/in/jesse-follet-phd/), I signed up for the [University of Washington's Python
Programing certificate](https://www.pce.uw.edu/certificates/python-programming) in October 2022 and finished the last
class in June 2023.

### Courses

1. [Essential Programming in Python](https://www.pce.uw.edu/courses/essential-programming-in-python)
2. [Advanced Programming in Python](https://www.pce.uw.edu/courses/advanced-programming-in-python)
3. [Web & API Development With Python](https://www.pce.uw.edu/courses/web-api-development-with-python)

### Learnings

I learned so much in this program that I was immediately able to put to work in my daily tasks. Some of the key things
that I learned were:

- Using standard Python libraries
- Package management
- Object-oriented programing
- Using and deploying [Django](https://www.djangoproject.com/) and [FastAPI](https://fastapi.tiangolo.com/) apps
- Testing with [unittest](https://docs.python.org/3/library/unittest.html)
  and [PyTest](https://docs.pytest.org/en/stable/)
- [Debugging with PyCharm](https://www.jetbrains.com/help/pycharm/debugging-your-first-python-application.html)
- Advanced constructs like context managers, decorators, and comprehensions
- Working with [MongoDB](https://www.mongodb.com/), [MySQL](https://www.mysql.com/), [SQLite](https://sqlite.org/)
